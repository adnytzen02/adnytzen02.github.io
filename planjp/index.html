日本旅行行程範本

1. 出發機場交通資訊

  • 出發日期：2025/06/18
  • 機場與航廈：桃園國際機場 第1航廈
  • 航空公司 / 航班編號：星宇航空 JX800
  • 報到開放 / 截止時間：起飛前3小時開放 / 起飛前1小時截止
  • 海關注意事項：行動電源不可托運、液體不得超過100ml

2. 航廈報到櫃台地圖

桃園機場第1航廈地圖

建議提早抵達，確認登機門與免稅店位置。

3. 抵達成田機場須知

入境教學
  • 依照指示前往入境審查（Immigration）
  • 準備護照與入境卡（部分電子填寫）
  • 過海關時需申報貴重物品與現金
  • 常見問答：滯留天數、住宿地點、返程日期

4. Skyliner 搭乘資訊

路線與購票方式
  • 搭乘地點：成田機場第1/第2航廈 B1 Skyliner 月台
  • 購票處：Skyliner售票機 或現場窗口
  • 推薦線上預約平台：京成電鐵官方網站
  • 終點站：上野／日暮里，可接JR或地鐵

5. 飯店資訊

  • 飯店名稱：例：APA Hotel 上野站前
  • 地址：東京都台東區上野7-12-11
  • 入住 / 退房時間：15:00後 / 10:00前
  • 推薦路線：日暮里轉山手線 → 上野站步行5分鐘

6. 出國行李清單

證件與必備物
  • 護照（有效期6個月以上）
  • 身份證（回程備用）
  • 機票、保險、住宿資料備份
生活用品與衣物
  • 衣物（依天數＋1套）
  • 折傘、保暖衣、防曬
  • 盥洗用品、藥品、轉接頭
科技用品
  • 行動電源（容量合規）
  • 手機、充電線、插座轉接頭
  • 網卡或Wi-Fi分享器
